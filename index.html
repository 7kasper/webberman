webberman

                
a simple websocket api tester — built for json & restn.io

            

            

                
connect to any websocket server and watch json messages stream in. compose path-based requests like restn.io expects, save them as templates, and click to fire.

                
    
                    
  • messages — one-line previews; click to expand into pretty-printed json.
    • 
                    
  • composed mode — builds { "path": "...", "params": {...} } envelopes with typed parameters.
    • 
                    
  • templates — click to send, ✎ to load & edit.
    • 
                    
  • errors — replies with error or code >= 400 are flagged red.
    •
